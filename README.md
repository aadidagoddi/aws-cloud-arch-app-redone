# AWS Cloud Architect

AI-powered AWS cloud architecture visualization and management tool.

## Features

- Visual cloud architecture design with 3D visualization
- AI-powered architecture generation using Claude
- Interactive module configuration
- Test case simulation with animated data flow
- Real-time AWS service integration
- Project lifecycle management (New → In Progress → MVP → Running)

## Tech Stack

- **Frontend**: React + Vite
- **Desktop**: Electron
- **3D Graphics**: Three.js + React Three Fiber
- **State Management**: Zustand
- **AI Integration**: Anthropic Claude API
- **Cloud**: AWS SDK

## Getting Started

### Prerequisites

- Node.js 18+ and npm
- AWS Account
- Anthropic API key

### Installation

\`\`\`bash

# Clone the repository

git clone https://github.com/YOUR-USERNAME/aws-cloud-architect.git
cd aws-cloud-architect

# Install dependencies

npm install

# Start development server

npm run electron:dev
\`\`\`

### Building for Production

\`\`\`bash
npm run electron:build
\`\`\`
